# markdown_previewer
_A Markdown preview App built with ReactJS_

### Installation
markdown_previewer requires [Node.js](https://nodejs.org/) and Webpack v8+ to run.<br />


Install the dependencies and start the server.

```sh
$ cd markdown_previewer
$ npm install -d
$ npm run build
```
